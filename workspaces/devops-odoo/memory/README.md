# Memoria — DevOps Odoo

Cada archivo en esta carpeta corresponde a un día de actividad: `YYYY-MM-DD.md`

## Formato de entrada por deploy

```
### HH:MM — PR #<número> — <título>
- **Módulos:** <lista>
- **Archivos cambiados:** <N>
- **Decisión FASE 1:** APROBADO / CERRADO (motivo)
- **FASE 2 resultado:** ✅ OK / ❌ ERROR
- **Módulos actualizados vía RPC:** <lista con versiones>
- **Notas:** <cualquier anomalía o detalle relevante>
```
